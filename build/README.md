# Build Help

```bash
Usage: ./makebuild ...
Options:
    HELP: Show build usage information (current file).
    TEST: Test Firmware Build.
    BOOTLOADER: Build to be used with a Bootloader.
```

# Build Examples

## Project inside ASF

```bash
./makebuild
./makebuild HELP
./makebuild TEST
./makebuild BOOTLOADER
./makebuild TEST BOOTLOADER
```
